# AI Agents + Figma Canvas — Designer's Guide

A practical guide for UX designers on how to set up and use AI coding agents (Claude Code, Cursor, Codex, etc.) to design directly inside Figma using your actual components and design system.

## What's inside

A single self-contained HTML page covering:

- **The big idea** — how AI agents connect to Figma via MCP and what's actually possible
- **Key terms** — MCP, Code Connect, agents, and other jargon explained plainly
- **What you need** — prerequisites and permissions checklist
- **Permissions & stakeholders** — who needs to approve what before you start
- **Installation walkthrough** — terminal and Claude app setup, step by step
- **Your role as a designer** — how the designer/agent collaboration actually works
- **Code Connect** — mapping Figma components to production code components
- **Prompting guide** — how to write effective prompts, with a template and CLAUDE.md tips
- **Security** — GDPR, sandboxing, access controls
- **Useful links** — MCP docs, Figma plugins, and more

## Usage

Open `figma-agents-guide.html` directly in a browser — no build step, no dependencies.

## Who this is for

UX/product designers who want to use AI agents in their Figma workflow but aren't sure where to start or what's involved on the setup and permissions side.
